# OledImageTools

#### 介绍
为了让图片显示在0.96寸OLED上，需要将图片分辨率转换为128x64，并且bpp（像素深度）为1bit。该工具将图片转换后输出到头文件中，形式为128x8长度（页写）的无符号字符型数组，需要设置oled为页寻址模式。
